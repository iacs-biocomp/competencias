## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## ORM

La documentación de [typeORM](https://orkhan.gitbook.io/typeorm/docs)

## Installation

```bash
$ npm i #Instala las dependencias del proyecto (tener node instalado)
```

## Correr la aplicación:

```bash
# development
$ npm run start

# Para desarrollo (Se actualiza cuando guardas cambios en el codigo)
$ npm run start:dev

# Para producción
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
